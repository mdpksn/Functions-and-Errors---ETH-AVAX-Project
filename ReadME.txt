# Bank Smart Contract

This is a simple Ethereum smart contract named Bank that allows users to deposit and withdraw Ether. It includes functions to deposit funds into the contract and withdraw funds from the contract.

## Features
- Deposit Ether into the contract
- Withdraw Ether from the contract if the user has sufficient balance
- Check account balance
- Use assert(), require(), and revert() successfully in the contract

## Usage
1. Deploy the `Bank` contract to the Ethereum blockchain.
2. Use the `deposit` function to deposit Ether into the contract.
3. Use the `withdraw` function to withdraw Ether from the contract.
4. Use the `checkBalance` function to check your account balance.

## Requirements
- Solidity compiler version ^0.8.24
- Ethereum development environment
